Axiomatic Complex System Dynamics Simulator

🌟 What is FaLaw Theory?
FaLaw (秩法图) is a novel axiomatic framework for modeling complex social systems. Based on 8 core axioms, it describes the dynamics of eight fundamental elements (乾定, 射, 陷, 离, 界, 散, 换, 坤转) and their interactions, leading to emergent phenomena like symbiosis, power dynamics, and phase transitions (坤转).
