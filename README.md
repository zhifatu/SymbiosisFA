## 📖 概述

**秩法图（Symbiosis Field Algebra）** 是一个从中国哲学土壤生长出来的系统科学框架。

它试图回答一个问题：

> **当无数个体各自追求目标、彼此施加影响时，秩序如何涌现？共生如何可能？**

我们用**8个元素**刻画系统的底层结构，用**原力守恒**约束演化过程，用**临界点 R=0.5**标记量变到质变的分界线，用**6维可能性空间**描述系统的未来态，用**4种张力场**诊断系统的健康度。

这不是隐喻，这是**可计算、可验证、可校准**的数学体系。

---

## 🔬 核心发现

### 1. 八元素完备性
系统状态可由**乾定、射、陷、离、界、散、换、坤转**八个维度完整刻画。  
**没有第九个元素**——这是公理，也是三千次模拟的结论。

### 2. 原力局域守恒
每个元素**至少保留50%**自身原力，其余流向其他七元素。  
**每行和为1**——这是系统的“质量守恒定律”。

### 3. 临界点 R=0.5
**所有效应函数在 R=0.5 处分界**：
- R < 0.5：线性响应，量变阶段
- R = 0.5：临界点，系统最敏感
- R > 0.5：非线性响应，质变阶段

这是秩法图最重要的**数值发现**——也是东方“过犹不及”思想的数学形式。

### 4. 可能性空间
系统在**6个正交维度**上演化：
- **稳定性**：维持现状的能力
- **灵活性**：响应变化的能力  
- **多样性**：保持差异的能力
- **连接性**：建立关系的能力
- **新颖性**：创造新质的能力
- **效率**：转化资源的能力

每个元素都是这个六维空间中的一个**固定向量**——这是秩法图的“元素周期表”。

### 5. 张力场诊断
系统健康度由**四种张力**刻画：
- **结构张力**：分布不均衡程度
- **动力张力**：流动不均衡程度  
- **势能张力**：潜能释放程度
- **冲突张力**：相互抑制程度

**总张力 > 0.7** 时系统进入风险区，**总张力 > 0.9** 时濒临坤转。
