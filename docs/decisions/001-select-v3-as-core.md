# 决策记录 001：选择 V3_S型抑制作为秩法图数学核心

## 状态
✅ 已采纳（2026-02-13）

## 背景
秩法图理论需要定义「原力激发」与「环境压力」的数学关系。
研究阶段提出了 5 个候选理论版本（V1–V5）。

## 候选方案对比
| 版本 | 核心特性 | 韧性范围 | 高压行为 | 合理性分数 |
|------|------|------|------|------|
| V1_原始理论 | 纯双曲线 | 0.30–1.50 | ⚠️ 持续增长 | 0.52 |
| V2_有界适应 | 适应性上限 | 0.35–1.20 | ✅ 收敛 | 0.68 |
| **V3_S型抑制** | **心理学S曲线** | **0.40–1.05** | ✅ **明显衰减** | **0.81** |
| V4_毒物兴奋 | 低压力促进 | 0.45–1.30 | ⚠️ 回升 | 0.59 |
| V5_动态阈值 | 分阶段响应 | 0.38–1.15 | ✅ 衰减 | 0.73 |

## 决策理由
1. **哲学一致性**——S型曲线是「量变到质变」「边界突破」的精确数学表达
2. **生物学合理性**——符合压力与表现的耶克斯-多德森定律
3. **工程稳定性**——高压区明确衰减，不会出现反直觉增长
4. **参数可解释性**——s 参数直接对应「界」强度，k 参数对应边界陡峭度

## 影响
- `DataSource.compute_primal_excitation()` 使用 V3 公式
- 所有场的原力计算统一从此接口获取
- 其他版本保留在 `research/archive/` 作为理论发展史

## 被拒绝的方案
V1、V2、V4、V5——合理性分数低于 0.75，且存在特定缺陷。
