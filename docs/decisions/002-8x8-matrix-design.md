# 决策记录 002：8×8 流转矩阵的数值设计原则

## 状态
✅ 已采纳（2026-02-13）

## 设计约束
1. **每行和为 1** —— 原力守恒
2. **对角线 ≥ 0.5** —— 元素必须保持至少 50% 自我
3. **非负性** —— 流转比例 ∈ [0,1]
4. **稀疏性** —— 每个元素只与 3–5 个其他元素有强耦合

## 核心数值依据
| 元素 | 自我保留率 | 依据 |
|------|------|------|
| 乾定 | 0.70 | 目标的高度稳定性 |
| 陷 | 0.75 | 场的粘滞性 |
| 散 | 0.55 | 易扩散特性 |
| 坤转 | 0.58 | 变革倾向 |

## 关键耦合路径
乾定(0.12)→射 → 观察定向
陷(0.15)→离 → 逃离压力
坤转(0.15)→乾定 → 革命后新目标
